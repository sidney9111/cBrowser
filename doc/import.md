#### Python - 3种加载模块实现
##### 1，使用系统函数import()

```
stringmodule = _import('string')
```

##### 2，使用imp 模块

```
import imp
stringmodule = imp.loadmodule('string',*imp.findmodule('string'))
```

##### 3，使用exec

```
importstring = "import string as stringmodule"
exec importstring
```
